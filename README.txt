-- Activar Backend

1. 'cd backend' ---> para estar en LAirPrope\backend
2. 'venv\Scripts\activate' ---> para activar el entorno virtual, debe salir primero que la ruta un mensaje "(venv)"
3. 'pip install -r requirements.txt' --> para installar todas las dependencias de la API
4. 'uvicorn API:app --reload ---> para inciciar el servicio. Para comprobar si sirve usar "http://127.0.0.1:8000/docs"
